폰트픽 이용에 대해 궁금하신 점을 확인해보세요

원하는 폰트 카드의 '무료 다운로드' 버튼을 누르면 해당 폰트의 원본 배포처(제작사 페이지 등)로 이동합니다. 폰트픽은 다운로드 링크를 정리해서 안내하는 큐레이션 서비스이며, 실제 폰트 파일은 제작사가 배포하는 원본을 그대로 받으시게 됩니다.

폰트픽에 큐레이션된 폰트는 대부분 상업적 이용이 가능하지만, 폰트마다 세부 허용 범위(인쇄물, 웹사이트, 영상, 포장지, BI/CI 등)가 다를 수 있어요. 상단의 '라이선스 보기' 버튼에서 일반적인 사용범위를 확인하시고, 정확한 조건은 반드시 각 폰트 제작사의 원본 라이선스 페이지에서 다시 확인해주세요.

상단 검색창에 "따뜻한 카페", "귀여운 SNS 카드", "깔끔한 이력서"처럼 자연스러운 문장으로 입력해보세요. 분위기, 업종, 용도, 개성 등 자연어 키워드 1,000여 개를 인식해서 어울리는 폰트를 바로 추천해드려요.

두 가지 방법이 있어요. ① 상단 미리보기 입력창에 원하는 문구를 직접 입력하거나 '문구선택' 버튼으로 준비된 예시 문구를 골라 모든 카드에 한번에 적용할 수 있어요. ② 각 폰트 카드의 미리보기 텍스트를 직접 클릭해서 그 카드만 원하는 문구로 바꿔볼 수도 있어요.

마음에 드는 폰트에 좋아요를 누르면 전체 방문자의 좋아요 수가 합산돼서 '인기 폰트 TOP 10'에 반영돼요. 로그인 없이 바로 누를 수 있고, 다시 누르면 취소도 가능해요.

상단 메뉴의 '폰트 찾아주세요' 게시판을 이용해주세요. 폰트가 보이는 이미지를 올리고 간단한 설명을 남기면, 확인 후 어떤 폰트인지 댓글로 답변해드려요. 로그인이나 폰트 이름을 몰라도 괜찮아요.

'폰트 찾아주세요' 게시판 목록에서 내가 올린 글을 다시 클릭하면 확인할 수 있어요. 답변이 등록되면 상태 표시가 '대기중' → '검토중' → '답변완료'로 바뀌니 상태만 확인해도 진행 상황을 알 수 있어요.

제작사 배포처 사정으로 링크가 변경되거나 만료됐을 수 있어요. 페이지 하단의 '문의하기'로 폰트 이름과 함께 알려주시면 빠르게 확인 후 반영할게요.

찾으시는 답변이 없다면 공지사항을 확인해보시거나 문의해주세요.
